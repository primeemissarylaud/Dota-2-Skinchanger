## Overview

A sleek and modern customization tool for Dota 2 that lets you personalize your in-game experience with custom cosmetic items.

### Key Features

- ✨ **Real-time skin preview**
- 🎨 **Huge collection** of custom cosmetics
- ⚡ **Fast switching** between configurations
- 📁 **Profile system** (multiple loadouts)
- 🔄 **Auto-update** support
- 🎮 **Clean & intuitive interface**
- 🌙 **Dark mode** optimized

## Installation

1. Download the latest release from the **Releases** tab
2. Extract the archive to any folder
3. Run `Dota2Skinchanger.exe` as administrator
4. Launch Dota 2 and enjoy your customized experience
